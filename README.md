# Калькулятор фигур на Java

Простая программа для вычисления площади и периметра:
- Прямоугольник
- Круг

## Как запустить
1. Склонировать репозиторий
2. Открыть в IntelliJ IDEA
3. Запустить `Main.java`

## Структура
- `Main.java` — точка входа
- `Rectangle.java` — класс прямоугольника
- `Circle.java` — класс круга
